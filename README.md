# AI Booking Assistant 

Installation

```bash
git clone https://github.com/HansrajS1/AI-Booking-Assistant
cd AI-Booking-Assistant

python -m venv venv
venv\Scripts\activate

pip install -r requirements.txt

python -m streamlit run app/main.py
```
